# إذاعات المغرب — تطبيق أندرويد

تطبيق بسيط لبث إذاعات مغربية مباشرة، مبني بلغة Kotlin ومكتبة ExoPlayer (Media3)،
مع تشغيل في الخلفية وإشعار تحكم.

## المتطلبات
- Android Studio (أحدث إصدار مستقر)
- JDK 17

## طريقة الفتح والتشغيل
1. افتح Android Studio.
2. اختر **Open** وحدد مجلد المشروع `RadioMaroc`.
3. انتظر حتى ينتهي Gradle Sync تلقائيًا (سيقوم Android Studio بإنشاء ملفات
   gradle-wrapper المفقودة تلقائيًا عند أول فتح، أو يمكنك تشغيل
   `gradle wrapper` يدويًا إذا كان لديك Gradle مثبت).
4. اضغط Run ▶ لتشغيله على جهاز حقيقي أو محاكي.

## ⚠️ خطوة ضرورية قبل النشر
روابط بث الإذاعات (Stream URL) هي خدمات خارجية لا يملكها هذا التطبيق،
وقد تتغير أو تتوقف من طرف الإذاعة نفسها في أي وقت. **قبل نشر التطبيق
على المتجر يجب عليك:**

1. فتح كل رابط من ملف:
   `app/src/main/java/com/example/radiomaroc/Station.kt`
   في متصفح أو في VLC للتأكد أنه يعمل فعليًا.
2. استبدال أي رابط لا يعمل برابط بديل صحيح لنفس الإذاعة (يمكن الحصول
   على الروابط الرسمية غالبًا من الموقع الرسمي للإذاعة أو تطبيقها،
   أو من مزودي التجميع مثل Radio Garden / TuneIn، مع احترام حقوق البث).
3. تعديل قائمة `StationRepository.stations` بإضافة إذاعات أخرى بنفس
   الطريقة إذا رغبت.

## التوقيع والنشر على Google Play
1. من Android Studio: **Build > Generate Signed Bundle / APK**.
2. اختر **Android App Bundle (.aab)** (مطلوب حاليًا لأغلب التطبيقات الجديدة).
3. أنشئ مفتاح توقيع (keystore) جديد واحتفظ بنسخة آمنة منه.
4. ارفع ملف `.aab` الناتج إلى Google Play Console.
5. عدّل `applicationId` في `app/build.gradle.kts` إذا أردت اسم حزمة مختلف
   عن `com.example.radiomaroc` (يفضل استخدام نطاق خاص بك).
6. استبدل أيقونة التطبيق (`app/src/main/res/drawable/ic_launcher.xml`)
   بأيقونة احترافية خاصة بك (يمكن استعمال أداة Image Asset في
   Android Studio: File > New > Image Asset).

## هيكلة المشروع
```
app/src/main/java/com/example/radiomaroc/
 ├─ MainActivity.kt          # الواجهة الرئيسية وقائمة المحطات
 ├─ RadioPlaybackService.kt  # خدمة التشغيل في الخلفية + الإشعار
 ├─ Station.kt               # نموذج البيانات وقائمة الإذاعات
 └─ StationAdapter.kt        # عرض قائمة المحطات (RecyclerView)
```

## ملاحظة حول حقوق البث
تأكد من أن لديك الحق في إعادة بث/تجميع روابط الإذاعات المستخدمة
(أغلب الإذاعات تنشر روابط بثها للاستخدام العام عبر تطبيقات الاستماع،
لكن يُستحسن مراجعة شروط كل إذاعة أو التواصل معها قبل النشر التجاري).
